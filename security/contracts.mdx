## Instructions

| Instruction | Purpose | Who can call | Funds movement |
| --- | --- | --- | --- |
| `configure` | Create/update the global `Config` PDA (fees, policies, quote mints, resolution council) | Initial: a fixed expected initializer key; updates: current `config.authority` | Seeds rent + global vault |
| `nominate_authority` | Step 1 of admin transfer — set `pending_authority` | Current `config.authority` | None |
| `accept_authority` | Step 2 — pending admin accepts authority | `config.pending_authority` | None |
| `add_whitelist_market_creator` | Whitelist a market creator (PDA `["wl-mkt", creator]`) | `config.authority` | Authority pays rent |
| `remove_whitelist_market_creator` | Remove a whitelisted creator | `config.authority` | Rent refunded |
| `mint_no_token` | Create the NO outcome mint + metadata, mint fixed supply to vault, revoke mint authority | Creator, subject to market-creation policy | Mints NO supply to vault |
| `create_market` | Create the YES mint + `Market` PDA, mint initial reserves, revoke mint authority | Creator, subject to policy; `team_wallet` must match config | Mints YES reserves to vault |
| `swap` | Buy/sell YES or NO against the per-side AMM | Any user | Quote ⇄ vault; tokens ⇄ vault; fee → `team_wallet` |
| `add_liquidity` | Add quote-side liquidity | Any user; `team_wallet` must match config | Quote user → vault |
| `withdraw_liquidity` | Withdraw a recorded LP position | The LP user; market not completed | Quote vault → user |
| `claim_winnings` | Burn winning (or void both) tokens and pay proportional quote | Any user; market resolved/completed | Vault → user |
| `resolution` | Set the winning side and mark the market completed | `config.authority` (if threshold 0) or an M-of-N resolution council | None (sets flags) |

## Tokens & custody

| Item | Detail |
| --- | --- |
| Outcome tokens | Per-market YES and NO SPL mints; fixed supply; mint authority revoked to `None` after init |
| Quote asset | Per-market `quote_mint`: native SOL (when default pubkey) or a whitelisted SPL mint (e.g. a USD stablecoin) |
| Custody | A program-owned global vault PDA (seeds `["global"]`) holds reserves; payouts are program-signed per on-chain rules |
| Pricing | Per-side constant-product AMM (`x * y = k`) |
| Payout | `claim_winnings` burns winning tokens and pays `amount × quote_reserves / token_reserves`; a VOID outcome refunds both sides |
